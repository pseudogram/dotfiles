# Install

- Initialize all submodules
    ```sh
    git submodule update --init \
        .oh-my-zsh/custom/themes/powerlevel10k \
        dot-config/nvim \
        dot-tmux-themepack \
        fzf \
        bbg # remove this line if on a personal machine
    ```
- Run installers
    - `./ubuntuInstaller`
    - `./macOsInstaller`
